Degree

Definition: Degree represents the number of direct connections of one node with another in graph.

What it tells us: Nodes to a greater degree tell us that there are a lot of direct connections to other nodes in the graph. On the other hand, a lesser degree tells us that there are few direct connections between these nodes.

Visualization Context: The degree of each node is used to determine its size when generating the visualization. To prevent disproportionate views, a relative size is applied to each node.
